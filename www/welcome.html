What's New?

  • Per-App Tile Preferences: Customize tile appearance (colors, background) individually for each app.
  • Manual CSS Input and File Upload: Add custom CSS styles to Disco Launcher using manual input or by uploading CSS files.
  • RTL Support: Improved right-to-left layout support for languages like Arabic and Hebrew.
  • Reduced Motion Animations: Added accessibility options to reduce animations on the welcome page.
  • Auto Theme Option: Automatically switch between light and dark themes based on system settings.
  • Accent Color Tile Background: Updated background selection label from 'Transparent' to 'Accent color'.
  • New Icons: Added new icons, including Metro Weather, Microsoft Office apps, and updated versions of Discord, TikTok, Twitter, and WhatsApp icons.
